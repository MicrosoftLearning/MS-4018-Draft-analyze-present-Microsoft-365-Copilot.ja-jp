# MS-4018: Microsoft 365 Copilot でドラフト、分析、プレゼンテーションを行う

## コースの説明

このコースでは、PowerPoint、Word、Excel、Teams、Outlook など、Microsoft 365 アプリの一般的なプロンプト フローについて説明します。 また、Microsoft 365 Copilot Chat も紹介し、作業ベースと Web ベースのデータの違いについて説明します。 これらの演習は、[Microsoft Learn](https://learn.microsoft.com/en-us/training/paths/draft-analyze-present-microsoft-365-copilot/) で利用可能な [Microsoft 365 Copilot でのドラフト、分析、提示](https://learn.microsoft.com/) というタイトルのラーニング パスに含まれるコンテンツに基づいています。

 **注**:これらの演習を完了するには、適切な Microsoft 365 Copilot プランを選択する必要があります。
 - **個人向け:**
    - お客様が Microsoft 365 アプリで Copilot 機能を使用するには、Microsoft 365 Personal または Family サブスクリプションを購入し、Microsoft アカウントにサインインする必要があります。
  - **ビジネスおよびエンタープライズ ライセンス:**
    - Microsoft 365 ビジネスおよびエンタープライズのお客様は、Microsoft 365 Copilot Chat を追加料金なしで利用できます。
    - Microsoft 365 Copilot を購入するには、エンタープライズまたはビジネス向けの、条件を満たす Microsoft 365 プランが必要です。 詳細については、次のサイトの FAQ セクションの適格性の前提条件に関する質問を参照してください。[エンタープライズ](https://www.microsoft.com/en-us/microsoft-365-copilot/enterprise#FAQ) または [ビジネス](https://www.microsoft.com/en-us/microsoft-365-copilot/business#FAQ) のための Microsoft 365 Copilot。
